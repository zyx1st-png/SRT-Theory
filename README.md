# Selection-Reality Theory (SRT) — public reading repo

A structured, public copy of one independent researcher's theory, organized so that **a human can read it** and **a language model can load it as context**.

It exists for one purpose: to make this framework legible enough that others can understand it, criticize it, and build on it — and so that, given two researchers' frameworks side by side, an LLM can help translate between them.

## Read it

Start with [`INDEX.md`](INDEX.md) for the reading order.

## Use it as AI context

See [`FOR_AI_CONTEXT.md`](FOR_AI_CONTEXT.md) — a single entry point that orients a language model and tells it which files to load and how to use them.

## What this is, and isn't

- It is a **public summary layer**, meant to convey the shape, force, and limits of the theory.
- It is **not** a final or authoritative source, and it does not ask you to accept anything — only to be able to understand it and argue with it.
