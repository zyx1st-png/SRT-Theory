# Reading order

Start at the top; each step adds depth.

1. **One-page overview** — the shape and force of the theory in a single page.
   - [`theory/SRT_one_page_EN.md`](theory/SRT_one_page_EN.md) (English)
   - [`theory/SRT_one_page_CN.md`](theory/SRT_one_page_CN.md) （中文）
2. **Quick start** — [`theory/SRT_quick_start.md`](theory/SRT_quick_start.md). A longer walk-through, with plain-language definitions of the core terms (`Ĝθ`, `d-value`, `Ψf`, `T_dir`) and a "what SRT is not" table.
3. **Core reversals** — [`theory/SRT_core_reversals.md`](theory/SRT_core_reversals.md). The five conceptual moves at the heart of the theory, each paired with the misreading it invites and the boundary that blocks it.
4. **Critical FAQ** — [`theory/SRT_critical_FAQ_CN.md`](theory/SRT_critical_FAQ_CN.md). Thirteen of the hardest objections (falsifiability, panpsychism, relation to the Free Energy Principle, the "explains everything" risk, AI vs. consciousness). Currently in Chinese; an English version is planned.

## Deeper material

This repo is a readable public layer. The framework also has a formal core — axioms, equations, and canonical definitions of the core terms — kept in the author's fuller materials. Curated formal summaries can be added here later.

To extend it: add a readable file under `theory/`, list it above, and (if it helps an LLM) reference it in [`FOR_AI_CONTEXT.md`](FOR_AI_CONTEXT.md).
